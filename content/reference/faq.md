### [zh-Hans]

# 常见问题 / FAQ

这里汇总下载与语言展示相关的高频问题，可先自助排查。

## 为什么按钮会跳转到代理域名？
为了改善 GitHub 访问稳定性，GitHub 链接会自动加代理前缀。

## iOS 为什么没有 GitHub Release 下载？
iOS 大多数客户端通过 App Store 分发，页面按官方分发渠道优先。

## 为什么有些页面只显示一种语言？
页面使用显式三语块，系统会根据浏览器语言只显示对应块。

# 常見問題 / FAQ

這裡整理下載與語言顯示相關的高頻問題，可先自行排查。

## 為什麼按鈕會跳到代理網域？
為了提升 GitHub 存取穩定性，GitHub 連結會自動加上代理前綴。

### [en]

# FAQ

This page covers common issues around downloads and language rendering for quick self-check.

## Why do download buttons open a proxy domain?
GitHub URLs are proxied to improve access stability.

## Why no GitHub Release links for most iOS apps?
iOS clients are mainly distributed via the App Store, which is prioritized.

## Why is only one language visible per page?
Pages use explicit tri-language blocks and render based on browser locale.
