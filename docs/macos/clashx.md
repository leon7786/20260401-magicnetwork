# ClashX（经典但已停更）

## 简介 / Intro
ClashX 是早期 macOS 上非常流行的 Clash 图形客户端。

ClashX was one of the most popular Clash GUI clients on macOS in earlier years.

## 状态 / Status
- 已停更（Archived / Unmaintained）
- 建议新用户优先使用 Mihomo Party、Clash Nyanpasu 或 FlClash。
