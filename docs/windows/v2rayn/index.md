# v2rayN

## 简介 / Intro
v2rayN 是 Windows 平台经典的 v2ray/Xray 客户端，节点管理直观，适合自建节点和进阶配置用户。

v2rayN is a classic v2ray/Xray client on Windows, with straightforward node management for self-hosted and advanced users.

## 类型 / Category
- v2ray 传统节点型
