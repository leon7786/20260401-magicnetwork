# HiddifyNext

## 简介 / Intro
HiddifyNext 是以 sing-box/Xray 等为核心的全能代理客户端，支持多协议与多平台。

HiddifyNext is an all-in-one client powered by cores like sing-box/Xray with broad protocol support.
